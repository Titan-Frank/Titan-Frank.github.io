# Wenbo Wu GitHub Homepage

这是 Wenbo Wu 的 GitHub Pages 个人主页源码。

- 学术与项目结合的单页主页
- 响应式移动端适配
- 项目卡片、时间线、联系方式
- 统一内容配置，后续可继续扩展

## 文件说明

- `index.html`：页面结构
- `styles.css`：视觉样式和响应式布局
- `script.js`：把内容配置渲染到页面
- `site-data.js`：个人资料与项目内容

## 本地预览

运行：

```bash
python3 -m http.server 4173
```

然后访问：

[http://localhost:4173](http://localhost:4173)
